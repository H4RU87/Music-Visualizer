About

    A awesome project utilizing D3.js library and Web Audio API to play and
visualize your choice of music. The Web Audio API captures and converts
the music frequency to a data set that is rendered into D3 charts. I got a lot
of inspiration from other developers while creating this project.

    Feel free to visit my Github page to take a look and play with the source code. Last note, I do not own any of the music on this website. Have fun!
